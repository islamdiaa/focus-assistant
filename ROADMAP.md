# FocusAssist Roadmap

This document outlines planned features for upcoming releases. Priorities may shift based on usage patterns and feedback.

---

## V1.8 — Knowledge & Reading

The next release focuses on building a personal knowledge capture layer directly into FocusAssist, so links, articles, and notes live alongside your tasks instead of scattered across browser tabs and apps.

| Feature | Description |
|---------|-------------|
| **Read Later Pocket** | Save links with auto-fetched title/URL, tag them by topic, mark as read when done |
| **Reading Notes** | Add freeform notes and highlights to any saved link |
| **Daily Digest** | Surface unread links on the Today page as a "Reading Queue" section |
| **Obsidian Export** | Push reading notes and highlights to your Obsidian vault alongside task data |

---

## V1.9 — Focus Experience

Enhance the focus session with music, ambient sounds, and tighter task-timer integration.

| Feature | Description |
|---------|-------------|
| **Spotify / YT Music Integration** | Link a favourite playlist and play it during focus sessions directly from the app |
| **Focus Ambient Sounds** | Built-in lo-fi, rain, white noise, and café options as a fallback when no playlist is linked |
| **Notification Audio Playback** | Wire Web Audio API to the existing sound preference so timer completion actually plays the selected chime |
| **Task-Linked Pomodoros** | Pick a task when starting a timer; track cumulative focus time per task in Stats |

---

## V2.0 — Intelligence

Introduce AI-powered features that reduce friction between thinking and doing.

| Feature | Description |
|---------|-------------|
| **AI Task Breakdown** | Paste a goal or project description, get a structured set of subtasks generated via LLM |
| **Smart Scheduling** | Suggest optimal time slots for tasks based on your historical energy patterns and completion data |
| **Weekly Review AI Summary** | Auto-generate narrative insights from your week's stats, completions, and carry-overs |
| **Natural Language Task Input** | Type something like "call dentist tomorrow high priority" and have it parsed into a structured task |

---

## V2.1 — Collaboration & Sync

Move from single-device to multi-device, and open up sharing and calendar awareness.

| Feature | Description |
|---------|-------------|
| **Multi-Device Sync** | Cloud storage backend (S3 or Supabase) so data stays in sync across machines |
| **Shared Templates** | Export and import template packs to share workflows with others |
| **Calendar Integration** | Google Calendar sync to see events alongside tasks on the Daily Planner |
| **Mobile PWA Improvements** | Offline support, push notifications, and improved touch interactions |
